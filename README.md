# 🌍 AI-Based Environmental Mapping

## 📖 Overview

AI-Based Environmental Mapping is a Flask-based web application that provides users with environmental insights for smarter and eco-friendly travel. The application combines real-time weather information, air quality data, and interactive maps to help users choose safer and healthier travel routes.

The project demonstrates the integration of multiple APIs with a responsive web interface to deliver environmental data in an easy-to-understand format.

---

## ✨ Features

- 🌦 Real-time weather updates
- 🌫 Air Quality Index (AQI) monitoring
- 🗺 Interactive map with location selection
- 📍 Eco-friendly route recommendations
- 📱 Responsive design for desktop and mobile
- ⚡ Fast Flask backend
- 🎨 User-friendly interface

## ⚙ Installation

1. Clone the repository

```bash
git clone https://github.com/Harish-1006/AI-Based-Environmental-Mapping.git
```

2. Move to the project folder

```bash
cd AI-Based-Environmental-Mapping
```

3. Install dependencies

```bash
pip install flask requests
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
